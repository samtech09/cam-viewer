About SAM's cam-viewer

Introduction

cam-viewer is a basic interface to HTML5 Video. You need to create symlink of source video folder to /static/videos.

Building and Installing

After you've downloaded it, cd into the directory and compile it with go build and run it with ./cam-viewer. Once it's running, go to http://ip-of-RPi:8080 for the fun to begin.
